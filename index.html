Tips

  • If the chapters of your book are not labeled "Chapter..." or "CHAPTER..." Simply add "#" to the beginning of the chapter title.
  • If the .txt file includes a Table of Contents, it can be removed. The application will build a Table of Contents within the ePub. If the entries include "Chapter" or "CHAPTER," it will cause empty chapters to be created.
